仿真原理： 气体多变过程 p·Vᵏ = 常数。弹簧压缩量 f 时，V(f) = V₀ − A·f，气体压力 p(f) = p₀·(V₀/V)ᵏ， 弹簧力 F(f) = [p(f) − p_a]·A。压缩端（f > 0）刚度快速增大（硬化），伸张端（f < 0）刚度下降（软化）， 呈现典型非线性特性——对应教材图 7-45。加辅助气室后曲线整体变平缓，固有频率降低。
空气弹簧刚度特性曲线（完整模型 — 参考实例19）
教材实例19（肖启瑞《车辆工程仿真与分析》）的完整模型：有效面积 A 和有效容积 V 均随弹簧变形量 x 线性变化，能更真实地反映膜式/囊式弹簧的非线性硬化特性。
A = A₀ + β·x  |  V = V₀ − α·x  |  p = (p₀+p_a)·(V₀/V)^γ − p_a  |  F = p·A
压缩（x > 0）：容积减小→压力升高，同时有效面积增大 → 刚度非线性增大（硬化）。 气体压力 p₀、大气压 p_a 和多变指数 γ 取自上方参数区。
1 L = 1 000 000 mm³
参数说明（教材表19-1 商用车空气前悬架参考值）： V₀ = 7 800 000 mm³；A₀ = 38 100 mm²；α = 59 000 mm³/mm；β = 18.6 mm²/mm；γ = 1.33。 蓝色实线为完整模型（A、V 均变化），绿色虚线为简化模型（A 恒定，V = V₀ − A₀·x），金色点为静载工作点（x = 0）。
计算公式说明
式 7-32 — 静平衡位置空气弹簧刚度
c₀ = (k · p₀ / V₀) · A² + (p₀ − p_a) · (dA/df)
第一项：气体压缩产生的刚度；第二项：有效面积随行程变化的附加刚度。
当 dA/df = 0 时简化为：c₀ = k · p₀ · A² / V₀
式 7-33 — 静平衡位置振动固有频率
n₀ = (1/2π) · √[ k·p₀·A·g / ((p₀−p_a)·V₀) + (g/A)·(dA/df) ]
g = 9810 mm/s²；A = πD²/4；p₀、p_a 单位 N/mm²（= MPa）；V₀ 单位 mm³。
乘用车目标约 1 Hz；商用车约 1.2 ～ 1.5 Hz。频率越低，悬架隔振性越好。
非线性仿真公式（气体多变过程）
p(f) = p₀ · [ V₀ / (V₀ − A·f) ]ᵏ
F(f) = [ p(f) − p_a ] · A
f > 0（压缩）：体积减小 → 压力升高 → 刚度增大（硬化）。
f < 0（伸张）：体积增大 → 压力降低 → 刚度减小（软化）。
加辅助气室：V₀ 替换为 (1+α)·V₀，曲线变平缓，固有频率降低。
